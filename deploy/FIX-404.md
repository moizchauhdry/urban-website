# Live WordPress 404 or blank page

## Setup (same as `/urban-app/` worked)

Upload **all of `dist/`** to:

```text
/var/www/urbanelitelimo/connecticut-black-car-service/
├── index.html
├── assets/
├── .htaccess
└── urban-favicon.png
```

```bash
npm run build
rsync -avz --delete dist/ user@server:/var/www/urbanelitelimo/connecticut-black-car-service/
sudo chown -R www-data:www-data /var/www/urbanelitelimo/connecticut-black-car-service
```

## WordPress

1. Trash any page with slug `connecticut-black-car-service`
2. Remove old rules that pointed this URL to `/urban-app/`
3. **Settings → Permalinks → Save**
